Complete Card System - All 10 Colors

Every color from Primary to Denary with proper variations, tints for backgrounds, and shades for borders/depth.

Core Functional Colors (1-5)

1. Primary - Background Layer

Main background layer using --bg-primary-subtle with --border-primary-subtle. Hover: --bg-primary-hover and --border-primary-light. Active: --bg-primary-active.

Primary Badge

2. Secondary - Foreground/Surface

Foreground and surface accent layer using --bg-secondary-subtle. Perfect for elevated surfaces and content areas.

Secondary Badge

3. Tertiary - Borders/Structure

Borders and structural elements using --bg-tertiary-subtle. Essential for form elements, dividers, and layout structure.

Structure Badge

4. Quaternary - Accent (Calm Emphasis)

Calm accent layer using --bg-quaternary-subtle. Perfect for highlighting without overwhelming, info states.

Accent Badge

5. Quinary - Emphasis/Positive

Emphasis and positive tone using --bg-quinary-subtle. Used for success states, CTAs, and important positive actions.

Success Badge

Extended Semantic Colors (6-10)

6. Senary - Utility Overlay

Utility overlay layer using --bg-senary-subtle. Perfect for tool tips, utility menus, and overlay content.

Utility Badge

7. Septenary - Danger

Danger and error states using --bg-septenary-subtle. Essential for destructive actions, errors, and critical warnings.

Danger Badge

8. Octonary - Info (Blue)

Information and neutral positive states using --bg-octonary-subtle. Perfect for informational messages, help text, and guidance.

Info Badge

9. Nonary - Warning/Neutral Accent

Warning and neutral accent using --bg-nonary-subtle. Used for caution states, pending actions, and neutral emphasis.

Warning Badge

10. Denary - Text Anchor

Text anchor on light surfaces using --bg-denary-subtle. Essential for readable text on light backgrounds and content areas.

Text Badge
